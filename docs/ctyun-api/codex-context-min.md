# Codex Context Min

## 项目背景
- 本项目是新仓库重构，不是旧仓库直接改。
- 主界面坚持极简、地图中心的桌面工作台，不回退到传统任务面板。
- 静默巡检替代显式任务面板，主界面只保留状态与结果承载。

## 首版交互基线
- 默认截图优先，单点播放，不做全量视频墙。
- 地图主视图负责点位承载，右侧抽屉负责详情，底部异常条负责异常缩略切换。
- 调试与诊断信息只写本地日志，不进 UI。

## 后续 ACIS 内核职责
- token 获取与刷新
- 平台接口访问
- 坐标转换
- 预览地址生成
- 宿主页处理
- 日志封装

## 开发提醒
- 所有 CTYun 相关接入统一走 `Infrastructure/Integrations/Acis`。
- `UI` 不直接调用平台，不直接处理签名、解密、坐标转换或日志细节。
