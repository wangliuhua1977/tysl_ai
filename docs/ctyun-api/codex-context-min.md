# Codex Context Min

## 项目背景

- 本仓库是 `Tysl.Ai` 的重构版新仓库，不是旧项目直接搬迁。
- 产品定位固定为地图中心型桌面监控系统，不是后台管理系统。
- 主界面长期保持“地图主视图 + 右侧详情抽屉 + 底部异常缩略条”。

## 当前开发阶段

- 第 2 轮已完成结构纠偏。
- 点位主源语义已经改为平台接口，本轮用 `StubPlatformSiteProvider` 作为权威源占位。
- 本地 SQLite 只保存 `site_local_profile` 补充信息，不再保存完整平台点位主档。
- 编辑弹窗只允许编辑本地补充信息，不允许新增平台设备。
- 坐标规则为平台坐标优先，本地手工坐标兜底，都没有则不显示地图点位。

## 当前仍未接入

- 真实 ACIS 内核
- 真实 CTYun 平台接口
- 真实地图 SDK
- WebView2
- webhook

## ACIS 后续职责

- token 获取与刷新
- 平台接口访问
- 坐标转换
- 预览地址生成
- 宿主页处理
- 日志封装

## 开发提示

- 所有 CTYun / ACIS 相关接入统一落在 `Infrastructure/Integrations/Acis`。
- `UI` 不直接调用平台，不直接处理签名、解密、坐标转换或日志细节。
- 第 3 轮再把当前 Stub 平台设备源替换为真实 ACIS 内核接入。
